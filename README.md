To center an element both horizontally and vertically, apply the display: flex;, justify-content: center;, and align-items: center; properties to the element's parent container.

<b> How the CSS properties work </b>   <br>

<ul>
  <li>display: flex;: This property turns the parent element into a flex container, enabling the flexbox layout model for its direct children.</li>
  <li>justify-content: center;: This centers the flex items along the main axis (the horizontal axis, by default).</li>
  <li>align-items: center;: This centers the flex items along the cross axis (the vertical axis, by default). </li>
</ul>
